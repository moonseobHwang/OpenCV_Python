# Finding Lane Lines on the Road

This project detects lane lines using Python and OpenCV. A video of the end-result can be found at:

https://youtu.be/-Tm4H4CrKT0?t=58


## Dependencies
* Python 3.8.3
* NumPy
* OpenCV
* Matplotlib
* Pafy
